# Level 1 — Simple Slot Machine

This project demonstrates core slot mechanics implemented in TypeScript.

## Features

- 3 reels slot machine
- Random Number Generator (RNG)
- Win detection logic
- Payout table
- Balance system
- Bet validation
- Game Over state
- Reset functionality
- Spin locking (prevent double spin)

## Technologies

- TypeScript
- Vite
- HTML / CSS

## How to run

npm install  
npm run dev
